<h2 align="center"> MiHRK </h2>

## Usage

1. Install dependencies

    ```
    npm install
    ```

2. Replace `<DB_CONNECTION_STRING>` and `<DB_PASSWORD>` in `config.env` with the connection string and password of your MongoDB instance.

3. Start the server

    ```
    node server.js
    ```
